# React Bundlers

## Webpack

- Most commonly used
- Bundle size 133Kb

## Rollup

- Lots of config files and dependencies
- Bundle size 137Kb

## Parcel

- No config files nor dependencies
- Bundle size 3,2Mb

## Vite

- Needs a global dependency. Has a CLI and doesn't need config files nor local dependencies
- Bundle size 149Kb
